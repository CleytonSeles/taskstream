# TaskStream

Sistema de gerenciamento de tarefas.

## 🏗️ Arquitetura
- Frontend: Vue.js + Tailwind CSS
- Backend: Node.js (Fastify) + PostgreSQL
- Deploy: AWS Lambda (Serverless)
- Observabilidade: OpenTelemetry + Jaeger
